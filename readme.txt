learngit
============================
1.git init 初始化版本库(把当前目录变成Git可以管理的仓库)
2.git status 查看当前库的状态(是否有改动)
3.git add file 把文件添加到版本库,这样我们就可以通过Git追踪管理这个文件
4.git commit -m "describe what you did this time"
5.Git 跟踪管理的是修改而并非是文件,当文件修改后被添加到暂存区后,执行提交操作后,结果最终才会被修改,而文件被修改了但未被添加到暂存区,则执行提交也不会改变结果.
6.Git 撤销修改,有两种情况,一种只是在工作区做了修改还未添加,则用git checkout -- f
ile来撤销,另一种是已经添加到暂存区了,此时的撤销我们要执行两步,首先是撤离暂存区gi
t reset HEAD file 之后可以用git status查看下当前状态,这时就回到了我们第一种情况>了,直接按第一种再操作一次即可.(Ps.如果已经提交了,则用之前的版本回退来操作.
